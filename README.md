### 🚀 Featured Projects

📌 **CowTrack - Hybrid Re-ID & Tracking System (MVP)**

End-to-end Computer Vision pipeline for individual cattle identification and spatial tracking from aerial drone footage:
* **Object Detection:** Implemented YOLOv8 Medium in HD (1280px) with custom Non-Maximum Suppression (NMS) and geometric filters.
* **Biometric Re-ID:** Trained a ResNet18 feature extractor to generate 256-dimensional embeddings with L2 normalization.
* **Vector Search:** Integrated FAISS for high-speed cross-pose similarity matching.
* **Hybrid Tracking:** Developed a custom tracking algorithm (Hungarian Assignment) combining Spatial Anchoring and Cosine Similarity, achieving 0 ID switches.
* **Visual Stability:** Engineered Exponential Moving Average (EMA) smoothing for sticky, jitter-free bounding boxes.
